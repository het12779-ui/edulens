# EduLens — The Intelligent Study-Streamer

Turn a 1-hour lecture video or a 50-page PDF into an interactive, searchable
study guide: clickable chapter outline, auto-generated flashcards, a visual
knowledge graph of concepts, and a "chat with this document" feature.


## Stack
- Frontend: Next.js + Tailwind CSS
- Backend: FastAPI (Python)
- AI: Whisper (transcription), Groq/OpenAI (LLM), ChromaDB + LangChain (RAG)
- DB: Supabase (Postgres + Auth)

Setup instructions will be added as each piece is built (see commit history).
